# BITSTREAM IS HERE

- **top.bit** : Bitstream for Digilent Basys3 board.

# HOW TO RUN

Use Vivado to program your Basys3 using the bitstream file.  
In the "Flow Navigator": "Program and Device" -> "Open Hardware Manager" -> "Program Device" -> [your board].  
Select the path of the "top.bit" file.  
Click the "Program" button on the UI.

NOTE: If "Program Device" is not active, check that the Basys3 USB drivers have been correctly installed.
